# Shared — Общие ресурсы

Эта папка для файлов, доступных всем агентам:
- Notion-схемы баз данных
- Общие learning-файлы
- Шаблоны и референсы

## Настройка Notion-баз

После создания баз в Notion, добавь их ID сюда:

### notion-databases.md
```markdown
| База | ID |
|------|----|
| Контент-план | YOUR_ID |
| SwipeFile | YOUR_ID |
| Learning Log | YOUR_ID |
```
