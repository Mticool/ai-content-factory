## Советы

1. **Будь конкретен** — "пост для Telegram про ошибки" лучше чем "напиши что-нибудь"
2. **Указывай формат** — пост, рилс, карусель, статья
3. **Давай контекст** — для кого, какая цель, какой CTA
4. **Используй фидбек** — 👍 / 👎 / ✏️ чтобы система училась
