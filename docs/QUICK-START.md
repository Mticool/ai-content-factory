# ⚡ Быстрый старт (5 минут)

## 1. Установи

```bash
curl -sSL https://raw.githubusercontent.com/Mticool/ai-content-factory/main/install.sh | bash
```

## 2. Заполни бренд

Открой и заполни 3 файла:
- `agents/orchestrator/brand/profile.md` — кто ты
- `agents/orchestrator/brand/voice-style.md` — как говоришь
- `agents/orchestrator/brand/audience.md` — для кого

## 3. Запусти

```bash
openclaw start
```

## 4. Напиши боту в Telegram

Примеры первых запросов:

> Напиши пост для Telegram про 5 ошибок начинающих предпринимателей

> Сделай сценарий рилса про тайм-менеджмент

> Разработай контент-план на неделю

> Карусель на 7 слайдов: как выбрать нишу

## 5. Обучай систему

- Понравилось → 👍 (система запомнит)
- Не понравилось → 👎 (система избежит)
- Поправь → ✏️ (система создаст правило)

Чем больше фидбека — тем точнее результат.
